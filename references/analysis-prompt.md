# 深度分析 Prompt

> **占位文件** — 用户后续提供具体 prompt 内容后替换。

---

## 使用位置

深度分析模式（模式二）步骤 3b：对每条"待沉淀"记录重新抓取正文后调用。

**输入：**
- 原文正文（来自 Agent-Reach 抓取）
- 元信息：标题、来源、链接、已有摘要

**输出：**
- 深度分析全文（Markdown 格式，写入飞书文档正文）

---

## 临时占位 Prompt

用户提供正式 prompt 前，使用以下基础 prompt：

```
你是一位专业的信息分析师。请对以下内容进行深度分析，输出结构化分析报告。

【原文信息】
标题：{title}
来源：{source}
链接：{url}
正文：
{content}

【分析要求】
请从以下维度进行深度分析，输出 Markdown 格式报告：

## 核心观点
（提炼文章最重要的 3-5 个观点）

## 深度解读
（对核心观点进行深入分析，结合背景知识，提供洞察）

## 关键数据与事实
（列出文中重要的数据、事实、案例）

## 影响与启示
（该内容对行业/技术/产品的影响，以及可获得的启示）

## 延伸思考
（基于文章内容提出的进一步思考问题或值得关注的方向）

输出语言：中文
输出长度：800-1500字
```

---

## 替换说明

用户提供正式 prompt 后：

2. 将正式 prompt 填入本文件
3. 保留"使用位置"说明
